# انشاء و الانتقال في آن واحد

نستطيع انشاء و الانتقال الى الفرع الجديد باستخدام الامر `checkout` مع الخيار `-b` كالتالي:

<div dir="ltr">

```bash
git checkout -b newBranch
```

</div>

استعرض الفروع للتاكد من انك في الفرع الجديد:

<div dir="ltr">

```bash
git branch
```

</div>

</div>
