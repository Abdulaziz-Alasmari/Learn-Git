# انشاء و عرض tag

تستخدم tags لاعطاء اسم لاخر commit و ذلك لتسهيل الوصول لها، كاعطائها رقم للاصدر المشروع. نقوم بانشاء tags كالتالي:

<div dir="ltr">

```bash
git tag v1.0.0
```

</div>

عند اضافة tag فانه يكون لاحدث commit موجود. و لاستعراض tags الموجودة:

<div dir="ltr">

```bash
git tag 
```

</div>
